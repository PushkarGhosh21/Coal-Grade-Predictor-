System Documentation

Technical Architecture

The WCL-NCL Coal Grade Prediction System utilizes an ensemble Machine Learning approach to forecast coal quality grades based on historical mining data.

  • Algorithm: Voting Classifier (XGBoost + Random Forest)
  • Data Source: WCL-NCL Area Historical Logs
  • Input Features: Mining Area, Temporal Seasonality (Cyclical Encoding of Month/Day)

Usage Policy

This tool is intended for internal decision support only. Predictions should be verified against physical lab samples (Third Party Sampling) before finalizing commercial invoices.
